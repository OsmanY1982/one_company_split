# 加尼米德 · 总览

## 一、一颗"类行星"的卫星

加尼米德直径 5,274 km——比水星（4,880 km）大 8%——但因水星主要由铁和岩石组成（密度 5.43 g/cm³），加尼米德主要是冰和岩石的混合（密度 1.94 g/cm³），质量仅为水星的 45%。形象地说：加尼米德是两倍大的月球，由冰、水和岩石组成——是一颗"冰行星"的完美例子。

## 二、核心参数

| 参数 | 数值 |
|------|------|
| 距木星 | 1,070,400 km |
| 直径 | 5,274 km |
| 质量 | 1.48 × 10²³ kg |
| 密度 | 1.94 g/cm³ |
| 自转周期 | 7.15 天（潮汐锁定于木星） |
| 表面温度 | ~ -163°C（日均） |
| 表面年龄 | 古老：40 亿年+（暗区），较年轻（亮区 ~20 亿年） |
| 磁场 | 固有磁矩，强度约为地球的 1/1000 |
| 轨道共振 | 与欧罗巴、伊奥同属拉普拉斯共振（4:2:1） |

## 三、为什么这么大？

加尼米德可能是在与木星类似的原行星盘环境中形成的——绕原始木星旋转的气体和尘埃吸积盘。这一过程与太阳系行星的形成相似——这使得加尼米德的大小和质量达到阈值：足以在自身重力下自发分化为铁核、岩石地幔和冰壳——成为一颗真正的"行星胚胎"。

## 四、与水星的竞赛

加尼米德经常被称为"比水星大的卫星"。但它和完全分异的岩石行星完全不同。水星经过了一次巨大的碰撞，剥离了大部分地幔，留下了超大的铁核；加尼米德缓慢分异，形成了一个液态铁核——产生了固有磁场——这一过程只有在冰岩石比有利时才可能发生。它们是太阳系中硬币两面：一颗是失去冰壳的纯岩石世界（水星），另一颗是过多冰壳的冰-岩世界（加尼米德）。
