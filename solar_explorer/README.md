# 太阳系科普板块 · SOLAR EXPLORER

太阳能星谱探索系统 — 306 颗 IAU 已命名天体的完整科普百科。

## 模块结构

| 文件 | 行数 | 功能 |
|------|------|------|
| `__init__.py` | 5 | 板块入口 |
| `body_data_entries.py` | ~500 | 33 颗主要天体详细数据（太阳+9行星+5矮行星+19大卫星） |
| `body_encyclopedia.py` | ~160 | 数据加载器，306 天体全量组装 |
| `star_catalog_window.py` | ~230 | 星谱目录窗口（搜索/分类/跳转） |
| `body_detail_window.py` | ~290 | 天体详情窗口（渲染+科普+语音） |
| `voice_reader.py` | ~60 | macOS TTS 语音朗读模块 |
| `README.md` | — | 本说明 |

## 功能特性

- **星谱目录**：搜索 + 5 类 Tab（全部/恒星/行星/矮行星/卫星），左侧列表展示
- **天体详情**：paint_planet() 引擎渲染球体 + 物理参数卡片 + 概述/物理特征/探测历史/趣味事实
- **语音朗读**：macOS say 命令 + Tingting 中文语音，非阻塞播放
- **入口对接**：智能中心悬浮球第 8 颗星球 + 太阳系窗口跳转按钮

## 入口

1. 智能中心悬浮球 → 最外圈 "星谱探索"（uranus 风格，轨道 460）
2. 太阳系天文馆 → 左下 "📖 打开星谱" 按钮

## 数据来源

NASA、ESA、JAXA 公开科学资料，中文科普风格编写。
