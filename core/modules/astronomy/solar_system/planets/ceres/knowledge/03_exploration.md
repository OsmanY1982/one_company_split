# 谷神星 · 黎明号探测

## 一、首次环绕矮行星

NASA 黎明号（Dawn）于 2015 年 3 月进入谷神星轨道——这是人类航天器首次环绕一颗矮行星（早于新视野号飞越冥王星仅数月）。在此之前，黎明号已环绕灶神星（Vesta，2011~2012 年）——使它成为首个也是迄今唯一环绕过两颗地外天体的航天器。

黎明号的离子推进系统（氙推进剂）是深空探测的革命——它通过持续数月的小推力实现从灶神星到谷神星的轨道转移——化学火箭完全无法做到。

## 二、关键发现总结

| 发现 | 意义 |
|------|------|
| 奥卡托亮斑（碳酸钠） | 证实地下卤水和近期低温火山活动 |
| 阿胡拉山（水冰火山） | 冰-岩混合浆低温火山的新范例 |
| 表面本土有机分子 | 小行星带中存在原生复杂碳化学 |
| 全球水冰分布 | 极区永久阴影区确认水冰，低纬有羟基信号 |
| 重力数据暗示液态水残留层 | 可能仍是活跃的"海洋世界" |
| 活跃的大气/外逸层 | 偶尔检测到水蒸气释放事件（可能来自升华或冰火山） |

## 三、与灶神星的对比

黎明号同时访问了灶神星和谷神星——两个同在小行星带但性质完全不同的天体：

| 特征 | 灶神星 | 谷神星 |
|------|--------|--------|
| 直径 | 525 km | 940 km |
| 密度 | 3.46 g/cm³ | 2.16 g/cm³ |
| 组成 | 岩石（玄武岩）| 冰+岩石混合 |
| 表面年龄 | 古老（>40 亿年）| 局部年轻（数百万年）|
| 起源 | 岩浆分异的原始行星胚胎 | 未全分异的 icy planetesimal |
| 撞击 | 保留巨大南极撞击盆地 Rheasilvia | 有年轻亮斑火山口 |

这一对比是行星科学的活教科书：同样位于小行星带的两个天体，一个（灶神星）因早期加热完全分异——有铁核、地幔和地壳——成为"几乎的行星"；另一个（谷神星）保留了水冰从而维持未完全区分——展示了初始物质条件（冰与岩的比例）如何根本性地决定行星演化的路径。

## 四、未竟的探测

黎明号在耗尽姿态控制燃料后于 2018 年 11 月结束任务——现在漂浮在谷神星周围的稳定轨道中，作为谷神星的人造卫星永久沉睡。

谷神星重返作为优先探测目标——因为它是太阳系中距离最近且最易到达的矮行星和冰世界。中国已表态有意在未来深空任务中纳入谷神星飞越。一次未来着陆器可以直接取样奥卡托亮斑的盐沉淀——如果发现微生物或有机结构，它将确立谷神星作为内太阳系天体生物学核心的地位。
