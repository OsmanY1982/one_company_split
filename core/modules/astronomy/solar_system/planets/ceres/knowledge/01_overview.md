# 谷神星 · 总览

## 一、基本身份

谷神星是小行星带中最大的天体，也是内太阳系唯一的矮行星。1801 年元旦，意大利天文学家皮亚齐（Giuseppe Piazzi）在巴勒莫天文台发现——这是人类发现的第一颗小行星。它最初被归类为行星，后在 1850 年代降格为小行星，最终在 2006 年（与冥王星同次 IAU 投票）升格为矮行星——是唯一一颗获得此"双升"的天体。

**为什么是矮行星而非小行星？** 谷神星的直径（约 940 km）和质量足够大，使其自身重力克服了刚性力，塑造成近似球体（流体静力平衡形态）——这是行星/矮行星与小行星的核心分界标准。

## 二、核心参数

| 参数 | 数值 |
|------|------|
| 距太阳平均距离 | 4.14 亿 km (2.77 AU) |
| 直径 | 939.4 km |
| 质量 | 9.38 × 10²⁰ kg（小行星带总质量的 ~25%）|
| 密度 | 2.16 g/cm³ |
| 表面重力 | ~0.27 m/s² |
| 逃逸速度 | 0.51 km/s |
| 轨道周期 | 4.6 年 |
| 自转周期 | 9.07 小时 |
| 表面温度 | -105°C (168 K) |

## 三、组成与内部

谷神星的低密度（2.16 g/cm³）揭示了其内部成分——约 25% 是水冰（远超一般小行星）。结构上推测分为：

1. **外壳（~40 km）**：水冰 + 黏土矿物 + 盐类的混合物——冰岩壳
2. **过渡层**：冰与硅酸盐的混合
3. **核心**：可能由含水硅酸盐矿物构成的非完全分异核

重力测量暗示谷神星内部可能仍残留一层液态水或泥浆层——约数十至数百公里深——这是太阳系中距离太阳最近的"海洋世界"候选者。

## 四、小行星带中的巨人

谷神星尽管只有月球的约 27% 直径，但其重力主导了小行星带的神州动力学——它产生的轨道共振与木星一起雕刻了小行星带中数千亿个天体的分布。它的轨道与许多小行星族共享，在历史上被认为是行星形成的"中止体"——一颗因木星的引力扰动而未能完成吸积的原始行星胚胎。
