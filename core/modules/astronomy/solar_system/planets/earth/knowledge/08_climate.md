# 地球 · 气候系统

## 一、碳循环：行星恒温器

地球气候在数十亿年间保持液态水稳定存在的核心机制是碳酸盐-硅酸盐循环：

```
大气 CO₂ + H₂O → H₂CO₃（碳酸）
H₂CO₃ + CaSiO₃（硅酸钙岩石）→ CaCO₃（碳酸钙）+ SiO₂ + H₂O
CaCO₃ → 板块俯冲带入地幔 → 数百万年后火山释放 CO₂
```

这一循环的速度取决于温度——温度越高，化学反应越快，风化速率越高，CO₂ 移除越快（负反馈）。这使地球避免陷入金星式的失控温室效应。

## 二、冰期与间冰期

过去 260 万年（第四纪），地球进入了"冰室气候"——被周期性的冰川扩展和退缩所主导。冰期-间冰期周期约 10 万年（过去 80 万年），与米兰科维奇旋回同步：

- **偏心率**（10 万年、40 万年周期）：地球绕日轨道形状的变化
- **地轴倾角**（4.1 万年周期）：22.1°~24.5° 之间变化
- **岁差**（1.9~2.3 万年周期）：地轴自转轴的进动

然而，米兰科维奇强迫引起的太阳辐射变化极小（高纬度夏季仅数 W/m²），冰期的大幅温度变化（~5°C）需要通过冰反照率和 CO₂ 的正反馈放大。南极冰芯恢复的 80 万年 CO₂ 记录揭示：CO₂ 和温度在冰期-间冰期尺度上几乎同步变化——CO₂ 既是"跟随者"又是"放大器"。

全新世（过去 11,700 年）是一个"异常稳定"的间冰期——它使农业和文明的诞生成为可能。

## 三、当前气候变化

自 1850 年以来，全球平均温度上升了约 1.3°C，其中约 1.1°C 明确归因于人类温室气体排放（IPCC AR6）。

**关键指标**：
| 指标 | 工业革命前 | 当前（约 2025）| 变化 |
|------|-----------|---------------|------|
| CO₂ | 280 ppm | 426 ppm | +52% |
| CH₄ | 722 ppb | 1,920 ppb | +166% |
| N₂O | 270 ppb | 337 ppb | +25% |
| 全球均温 | 基准 | +1.3°C | — |
| 北冰洋夏季冰面积 | ~1,000 万 km² | ~400 万 km² | -60% |
| 海平面（自 1900）| 基准 | +20 cm | 加速上升 |
| 海洋 pH | 8.2 | 8.05 | -0.15（+30% 酸度）|

## 四、翻转点（Tipping Points）

多个地球子系统可能在特定升温阈值被越过"翻转点"——不可逆跃迁为截然不同的状态：

| 系统 | 可能翻转点 | 影响 |
|------|-----------|------|
| 格陵兰冰盖 | +1.5~3°C | 7 m 海平面上升 |
| 大西洋环流（AMOC）| +2~4°C | 北半球气候剧变 |
| 亚马孙雨林 | +3~4°C + 砍伐 | 转变为稀树草原，释放碳 |
| 永冻土 | +1~2.5°C | 释放 CH₄ 和 CO₂ |
| 珊瑚礁 | +1.5°C | 全球 70~90% 退化 |

翻转事件可能级联——一个系统的崩溃触发下一个。

## 五、古气候：雪球地球

约 7.17~6.35 亿年前的成冰纪，地球可能至少两次完全被冰覆盖——"雪球地球"事件。赤道地区被厚达 1 km 的冰盖覆盖，全球平均温度约 -50°C。生命退守至热液喷口和冰裂缝的微小避难所。火山活动持续排放 CO₂（因冰封无风化移除），经过数千万年积累，温室效应最终打破冰球——随即触发了埃迪卡拉生物群和寒武纪大爆发。雪球地球暗示生命的韧性可达极端，但也提醒气候系统翻转后的代价之巨大。
