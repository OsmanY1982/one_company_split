# 恩克拉多斯 · 总览

## 一、被忽略 225 年的小点

1789 年，威廉·赫歇尔（William Herschel）用他自制的 1.2 m 望远镜发现了这颗暗淡的卫星。接下来 225 年它只是土星卫星列表中的一个小条目。直到卡西尼探测器抵达土星系统——恩克拉多斯瞬间成为举世关注的焦点。

## 二、核心参数

| 参数 | 数值 |
|------|------|
| 距土星 | 237,948 km |
| 直径 | 504 km（仅比英国大一点）|
| 质量 | 1.08 × 10²⁰ kg |
| 密度 | 1.61 g/cm³ |
| 反照率 | 0.99（太阳系最高——近乎完美反光镜）|
| 表面温度 | ~ -198°C（75 K），南极少部分 > -120°C |
| 自转周期 | 1.37 天（潮汐锁定于土星） |
| 内部 | 全球地下海洋，厚冰壳，多孔岩石核心 |

## 三、反照率 0.99——太阳系最亮的天体

恩克拉多斯表面几乎完全由纯水冰晶体覆盖——巨大的冰粒不断从羽流中沉降到表面，覆盖了任何灰尘或撞击产物。这颗小小的卫星是一面不断更新的完美反光镜。加上它的体积小，无法保留任何大气（逃逸速度仅约 0.24 km/s），没有雾霾或吸收层——反射了几乎全部入射阳光。

## 四、轨道共振驱动

恩克拉多斯与土星第四大卫星迪奥内（Dione）处于 2:1 轨道共振——迪奥内每绕一圈，恩克拉多斯绕两圈。共振使恩克拉多斯轨道保持微弱椭圆偏心率——因此在不同阶段经历不同的土星引力——这种"引力的周期性伸缩"产生了潮汐加热，集中在南极高纬度区。热量驱动冰壳开裂——水羽流从中喷出——每月向土星环 E 环补充数吨冰粒。
