# 火星 · 卫星：火卫一与火卫二

## 一、Fire Star's Sons

火星两颗卫星以战神的双马命名——Phobos（恐惧）和 Deimos（恐怖）。两者均极小、极不规则、极暗（反照率仅约 0.07）。

| 参数 | 火卫一 Phobos | 火卫二 Deimos |
|------|--------------|--------------|
| 直径 | 22.2 km | 12.6 km |
| 轨道高度 | 6,000 km | 20,068 km |
| 轨道周期 | 7h 39m | 30h 18m |
| 密度 | 1.88 g/cm³ | 1.47 g/cm³ |
| 基本类型 | 碳质小行星类 | 碳质小行星类 |

## 二、起源之谜

两颗卫星的外观和光谱类似 C 型小行星——长期以来被认为是被火星引力捕获的主带小行星。然而，它们的轨道近乎圆形、几乎精确位于火星赤道面上——这两个特征对被捕获天体来说极不自然（捕获应产生高偏心率、高倾角轨道）。

替代理论：
- **大撞击**：一个类似月球形成的巨撞击从火星抛出碎屑盘，Phobos 和 Deimos 从中吸积
- **碰撞破碎**：一颗被捕获的小行星后来在潮汐力下碎裂为两星，轨道圆化

## 三、火卫一的命运

火卫一正在螺旋下坠。它位于火星同步轨道（约 17,000 km）之内，火星的潮汐力使其以每百年约 1.8 m 的速度向内螺旋。约 3,000~7,000 万年后，它将越过火星的洛希极限——被潮汐力撕裂，碎屑形成环绕火星的岩石环——或直接撞入大气中被摧毁。

火卫一的搏命螺旋使其成为研究行星潮汐演化的天然实验室。其表面覆有深达数百米的细粉层——来自微陨石撞击的累积。

## 四、探测未来

苏联 Phobos 1/2（1988）试图着陆火卫一但失败。ESA 的 Phobos-SOHO 等概念持续多年但未获批。日本的 MMX（火星卫星探测）任务计划于 2020 年代晚期发射——将首次从一颗火星卫星采集样品返回地球。
