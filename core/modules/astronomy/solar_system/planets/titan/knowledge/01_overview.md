# 泰坦 · 总览

## 一、另一个"地球"

1655 年，荷兰天文学家 Christiaan Huygens 首次识别出土星最大的卫星。但直到 1980 年旅行者 1 号飞越，泰坦才从"一个暗淡的光点"升级为一个真实世界。浓密的橙黄色大气完全遮挡了表面——旅行者的图像只看到一层弥漫的烟雾球。

卡西尼-惠更斯任务（2004~2017）彻底改变了对泰坦的认知：这不是冰卫星——这是一个拥有液态湖泊、河流三角洲、沙丘海洋、降雨和季节循环的活世界。只是它的液体不是水——而是液态甲烷和乙烷（在 -180°C 下稳定）。

## 二、核心参数

| 参数 | 数值 |
|------|------|
| 距土星 | 1,221,870 km |
| 直径 | 5,150 km（比水星大） |
| 质量 | 1.345 × 10²³ kg |
| 密度 | 1.88 g/cm³ |
| 表面温度 | ~ -179°C (94 K) |
| 大气压（表面）| 1.45 地球大气压（1.47 bar） |
| 大气组成 | 95% N₂, 5% CH₄, 微量 H₂ 和碳氢化合物 |
| 自转周期 | 15.9 天（潮汐锁定于土星） |
| 表面特征 | 液态甲烷/乙烷湖泊海、有机沙丘、水冰基岩 |

## 三、泰坦的独特地位

泰坦在太阳系中是唯一的——没有任何其他卫星、矮行星或小行星拥有可视为"大气"的稠密气体层。它之所以能保留浓密大气，是因为：
1. 温度极低（94 K）——气体热速度低，难以逃逸
2. 质量足够大——逃逸速度约 2.64 km/s
3. 远距离——太阳风在此处已减弱

土星磁层实际为泰坦提供了额外的大气保护伞——泰坦大部分时间在土星磁层内运行——免受太阳风剥蚀。
