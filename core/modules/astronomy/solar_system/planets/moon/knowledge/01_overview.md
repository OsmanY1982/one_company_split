# 月球 · 总览

## 基本身份

月球是地球唯一的天然卫星——太阳系第五大卫星。它是除了地球以外人类唯一踏上过的天体。地月系统有时被称为"双行星"——月球相对于地球的大小（直径比 0.273）在太阳系行星-卫星对中无出其右（仅冥王星-卡戎比更大，但冥王星已被降级为矮行星）。

## 潮汐锁定

月球永远以同一面朝向地球——近地面——因为月球已被地球的潮汐力完全锁定。由于天平动（因轨道椭圆和月球轴倾角造成的视摆动），从地球上实际可观测到约 59% 的月球表面——而非正好 50%。月球背面从未被直接看到，直到 1959 年苏联月球 3 号传回影像——揭示了一个与近地面截然不同的、环形山更密集、几乎无月海的世界。
