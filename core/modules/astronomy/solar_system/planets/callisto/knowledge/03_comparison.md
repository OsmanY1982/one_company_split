## 二、潮汐加热递减梯度

从伊奥到卡利斯托：与木星的距离每倍增，潮汐加热减少约 100 倍。这四颗卫星恰好展示了潮汐加热如何塑造一个天体：

1. **伊奥（最近）** → 潮汐加热 ≈ 10¹⁴ 瓦 = 全球熔岩世界
2. **欧罗巴** → 潮汐加热 ≈ 10¹² 瓦 = 冰下海洋，冰构造运动
3. **加尼米德** → 潮汐加热 ≈ 10¹⁰ 瓦 = 微弱构造，微小磁场对流
4. **卡利斯托（最远）** → 潮汐加热 ≈ 0 = 原始化石

## 三、木星系统的层次宇宙

木星及其伽利略卫星构成了一个微缩太阳系——大质量中心天体（木星=太阳）周围轨道运行的四个世界各自独立演化。这一系统在 1610 年直接启发了哥白尼太阳系模型——因为看到了绕木星运行的卫星，证明了天体可以绕非地球中心运行。近 400 年后的探测器证实：这种微缩类比比伽利略想象的要深刻得多——每个伽利略卫星上的条件模拟了行星演化的不同阶段和不同的地质引擎。
