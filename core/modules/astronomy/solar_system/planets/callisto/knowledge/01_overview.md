# 卡利斯托 · 总览

## 一、被遗忘的巨兽

卡利斯托是伽利略四卫星中最外层、最不被关注的一颗——它距木星约 188 万 km，绕木星一圈需 16.7 天。它不在拉普拉斯共振中——因此不受潮汐加热——没有火山、没有构造活动、没有大气。它是给伽利略卫星中唯一保持原始面貌的天体——未曾被内部过程改造。

## 二、核心参数

| 参数 | 数值 |
|------|------|
| 距木星 | 1,882,700 km |
| 直径 | 4,821 km（非常接近水星） |
| 质量 | 1.08 × 10²³ kg |

| 自转周期 | 16.7 天（潮汐锁定） |
| 表面温度 | ~ -139°C（日均） |
| 表面年龄 | > 45 亿年（太阳系最古老的表面之一） |
| 表面特征 | 密集撞击坑，几乎无地表构造 |

## 三、古老的冻结表面

卡利斯托表面是太阳系中最饱和的撞击坑记录——没有一块区域不笼罩在密密麻麻的撞击坑之中。暗淡的黑色表面是亿万次微撞击混合和分子破碎（空间风化）的结果——与月球高地极其相似。

卡利斯托证明了在缺乏内部地质改造的世界中，撞击会在数十亿年时间尺度上将表面磨碎成极细粉尘——也称为风化层（regolith）。卡利斯托的风化层厚度估计超过 100 m——是太阳系中已知较厚的之一。

## 四、为什么没有内部活动？

关键原因：卡利斯托不在轨道共振中。伊奥、欧罗巴和加尼米德的 4:2:1 拉普拉斯共振迫使它们做椭圆轨道运动，但卡利斯托从未被纳入——它的潮汐加热大约为零。无线电测距表明卡利斯托内部未完全分异——冰和岩石仍混合在一起，没有形成明显的铁核。换句话说，卡利斯托是一块"冻结的化石"——记录着太阳系早期行星胚胎在未加热状态下的原始结构。
