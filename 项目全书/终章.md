## 技术栈总览

| 层级 | 技术 | 说明 |
|------|------|------|
| 界面框架 | PyQt5 | GUI 框架，QPainter 手绘渲染 |
| 数据库 | SQLite 3 | 多库架构，每个域独立 .db |
| AI 后端 | Ollama / OpenAI / DeepSeek / Claude / 通义千问 | 6 后端统一 LLMClient 封装 |
| 语音识别 | faster-whisper (CTranslate2) | 离线语音识别，tiny/base/small 三档 |
| 扫码 | pyzbar + Pillow | 条码/二维码识别 |
| 密码安全 | bcrypt (cost factor=12) | 密码哈希和验证 |
| 图表 | Matplotlib | 数据可视化绑定到 PyQt |
| 依赖管理 | 自建 deps.py | wheel 离线安装，不走 pip |
| 打包 | PyInstaller | macOS .app 分发 |
